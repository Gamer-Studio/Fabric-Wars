# Fabric-Wars

## 개발 환경
> engine - Unity 2021.3.21f1
> platform - windows, android?

## 게임 디자인
> 싱글 & 멀티 플레이어
> 2d 탑뷰
> 청크기반 무한 월드

## 게임 진행 구상
세계의 엔티티와 상호작용하여 원소 & 아이템을 수집
아이템을 연소시켜 원소를 추가로 획득하거나 아이템의 기능으로 원소를 추가적으로 수집
원소를 소모하여 무작위로 특수한 엔티티를 소환하거나 특수한 능력을 사용

## 게임 최종 목표
월드 내 모든 지역을 탐사하고, 각 지역의 보스 몬스터 처치
